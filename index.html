Automated Irrigation System

Problem Statement

"Irrigation of Fields based on Crop Requrirements"

Technology Used

  • IoT sensors : DHT11, Soil Moisture, Relay, NodeMCU
  • Machine Learning(SVM)
  • Flask
  • Python
  • ArduinoIDE
  • Firebase

Description

  • This system automatically irrigates fields based on crop type using Machine Learning Models.
  • NodeMCU collects data like moisture, temperature and humidityfrom soil and environment and sends it to Firebase.
  • The pre-trained ML model uses these readings to decide whether to turn the water pump on or off.
  • The ML model in Cloud then sends the irrigation recommendations back to the NodeMCU through Firebase.

Accident Alert System

Problem Statement

"Accident Alert System(For Bike)"

Technology Used

  • Arduino UNO
  • SIM800L
  • Neo 6M GPS
  • Stroke Sensor
  • ArduinoIDE

Description

  • If an accident occurs, the system sends a message to the SOS contact with the coordinates of the accident location.
  • If the stroke sensor senses a stroke beyond a certain limit, the system starts beeping for 30 seconds.
  • If the bike falls accidentally, the user can reset the module within 30 seconds.
  • If the module is not reset after 30 seconds of a fall, it transmits an SMS with the coordinates to the SOS contact.
